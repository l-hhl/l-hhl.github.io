    • #1. CSS简介

    1.1 HTML的局限性

    它只关心内容的语义，不关心整体页面

    1.2 CSS 网页美容师

    CSS是层叠样式表的简称，有时我们也会称为CSS样式表或级联样式表。

    CSS也是一门标记语言

    CSS主要修饰细节，布局，外观样式，美化html。

    最大价值，由HTML专注去做结构呈现，样式交给CSS，即结构（HTML）与样式（CSS）相分离

    1.3 CSS

    css规则由两个主要的部分构成，选择器以及一条或多条声明。

    给谁改样式{改什么样式}

    选择器是用于指定CSS样式的HTML标签，花括号内是该对象设置的具体样式。

    属性和属性值以”键值对”的形式出现

    注意结尾要加分号

    1.4 CSS代码风格

    ###1.4.1 代码风格 

    h3 { color: XXXXXX;font-size: 20px}

    称为紧凑格式 

    h3 {
     color: XXX;
     font-size: 20px;
    }

    ###1.4.2 样式大小写

    大小写都能执行，但是一般为小写

    ###1.4.3 空格

    1.属性值前，，冒号后，加一个空格。

    2.选择器和大括号中间保留一个空格。

    #2.CSS基础选择器

    ##2.1 CSS选择器的作用

    选择器或者选择符就是根据不同需求把不同的标签选出来。

    ##2.2 选择器分类

    选择器分为基础选择器和复合选择器两个大类。

    基础选择器是由单个选择器组成的。

    基础选择器又包括：标签选择器，类选择器，id选择器和通配符选择器。

    ##2.3 标签选择器
    标签选择器是指用HTML标签名称作为选择器，按标签名称分类，为页面中某一类标签指定同统一的CSS样式。

    它能够将所有的标签选择出来。

    快速，但不能差异化设置。

    ##2.4 类选择器
    类选择器能够实现差异化选择。

    要用到class属性实现差异化选择。 class可选择一个或多个标签。

    语法：
    .class {
    属性值： 属性值1
    …..
    }

    注意，有 **.**

    不能用标签名做类名，长名词加-来分隔。不要使用纯数字，中文，命名尽量要有意义。尽量使用前端开发规范手册。

    类选择器画盒子（div）

    类选择器的多类名：
    我们可以给一个标签指定多个类名，从而达到更多的选择目的，这些类名都可以选出这个标签。

    多类名的使用：在同一个class的引号内，中间记得加上空格。

    可以把一些标签元素相同的样式放入一个类里面，这些标签都可以调用这个公共的类，然后再调用自己独有的类。从而节省CSS代码，统一修改也非常方便。

    2.5 id选择器

    语法：

    #id名{
        属性1： 属性值1；
    }

    id调用只能调用一次，id属性只能在每个HTML文档中出现一次。

    id选择器和类选择器的区别

    类选择器好比人的名字，一个人可以有多个名字，同时一个名字也可被多个人使用，id只能对应一个html标签，两个标签的id属性必然不相同，class则不然

    id选择器好比人的身份证，唯一的

    就是同一个 HTML页面，如果你有两个或以上的html元素的id属性一样，就会有冲突

    最大区别在使用次数上

    id选择器一般用于页面唯一性的元素上，经常和js搭配使用

    2.6 通配符选择器

    在CSS中，通配符选择器使用*定义，它表示选取页面中所有的标签
    语法：

    * {
        属性1： 属性值1；
    }

    通配符选择器不需要调用，自动就给所有的元素使用样式

    特殊情况下才使用

    3.CSS字体属性

    CSS fonts属性用于定义字体系列，大小，粗细，和文字样式

    3.1字体属性

    CSS使用font-family属性定义文本的字体系列

    各个字体之间用英文状态下的逗号隔开

    一般情况下，如果有空格隔开的多个单词组成的字体，加引号

    尽量使用系统默认自带字体，保证在用户的浏览器中都能正常显示

    谷歌浏览器默认字体，微软雅黑

    一般直接给body标签指定CSS字体属性。多个字体指定是注意系统的执行顺序。

    3.2 CSS字体大小

    font-size XXpx

    px像素大小是网页的最常用的单位

    谷歌浏览器默认的文字大小为px

    不同浏览器的默认文字大小可能不一样

    为了方便直接给body标签设置CSS样式

    一般标题需要单独指定文字大小

    3.3 字体粗细

    CSS使用font-weight属性设置文字粗细 属性过多直接去查找CSS手册去查找属性

    注： 700=bold normal=400（只变大小不改粗细）不加PX
    实际开发中更提倡数字

    学会让加粗标签不加粗

    3.4 文字样式 （style）

    normal ：默认值，显示正常字体， 让倾斜的不倾斜，如结合em标签使用

    italic：斜体

    3.5 字体复合属性

     h2 {
                    font: font-style font-weight font-size/line-heigh font-family;

            }

    必须按照这个顺序写，中间用空格分隔

    不需要的属性可以省略，但是size和family必须有

    4.CSS文本属性

    文本属性定义文本的外观，如颜色，对齐文本，装饰文本，文本缩进，行间距等

    color用于定义文本的颜色

    表示方法：预定义的颜色值，如：red，十六进制，如：#FF000,以及RGB代码，最常用的是十六进制 。

    ##4.2 对齐文本

    text-align：水平对齐，有居中，center left（左对齐） right（右对齐）

    4.3装饰文本

    text-decoration属性规定添加到文本的修饰，给文本添加下划线，删除线，上划线等。

    none：默认，没有装饰线，取消简介自带的下划线
    underline: 下划线，链接a自带下划线
    overline: 上划线
    line-through: 删除线

    重点了解如何添加下划线，以及如何删除下划线

    4.4文本缩进

    text-indent属性用于指定段落缩进

    单位：PX（像素） em（相对单位，就是当前元素一个文字的大小，如果当前元素没有设置大小，会按照父元素的一个文字大小。）（当前元素一个文字大小的单位）

    单位可以取负值，表现为突出。

    4.5 行间距

    line-height属性用于设置行间的距离（行高）可以控制文字行与行之间的距离。单位px。

    行间距包括：上间距，文本高度，下间距，三个部分。

    改变行间距真正改变的是上下行间距（文字大小不变的情况下）

    多出来的行间距（设置的减去文字大小）评分给上下行间距

    多行文本算行间距：第一行最下方算到第二行最下方

    ##5.1 内部样式表

    内部样式表或者说是内嵌样式表是写到html页面内部，是将所有的CSS代码抽取出来，单独放到一个style标签中 理论上 style标签可以放到任何地方，但一般放到head中，练习中常用。

    控制范围是整个页面，

    但是CSS与HTML没有完全分离。

    ##5.2 行内样式表 （内联样式表）

    行内样式表是在元素标签内部style属性中设置CSS样式，适合于简单修改样式。

    1.只控制当前标签

    2.并没有实现结构与样式相分离
